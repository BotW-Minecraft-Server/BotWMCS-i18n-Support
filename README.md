# BotWMCS-i18n-Support
国际化语言支持

## 目前没有完成国际化语言支持的模组有：
- Natural Decor Mod
- Bayou Blues
- Chisels and Bits
- More Furniture

## 目前完成国际化语言支持但完成度不高的有：
- 潘马斯系列
- MrCrayfish's Furniture Mod
- Quark

## 如何上传自己的国际化语言文本？
- 使用 Fork 工具将该 repo 复刻到自己的账户上
- 推荐使用 GitHub 桌面应用程序将其下载到本地
- 将自己的国际化语言文本上传至本地 repo，并使用 pull request 工具合并代码到该仓库
- 经过开发组审核通过后即可成功上传
